Le modèle Dynamic HTML Standard Edition produit un code HTML
qui utilise une feuille de styles en cascade (CSS) et est conforme à
la norme HTML 4.0. Dynamic HTML Standard Edition exige donc une
version 4 ou supérieure du navigateur.

Pour plus d'informations sur l'utilisation et la personnalisation
de projets créés à partir de ce modèle, reportez-vous à la section
Référence des modèles dans l'aide en ligne ou les fichiers PDF
fournis.
