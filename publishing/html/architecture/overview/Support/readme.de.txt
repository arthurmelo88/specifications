Die Schablone "Dynamic HTML Standard Edition" erzeugt
HTML-Code, der eine CSS-Datei (Cascading Style Sheet)
verwendet und mit der HTML-Standardversion 4.0 kompatibel
ist. Für Ausgaben mit "Dynamic HTML Standard Edition" ist
ein Browser ab Version 4 erforderlich.

Weitere Informationen zum Verwenden und Anpassen von
Projekten, die mit dieser Schablone erstellt wurden, finden
Sie im Abschnitt "Schablonen" der Online-Hilfe bzw. in den
mitgelieferten PDF-Dateien.
